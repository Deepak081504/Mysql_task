Explanation:
        Retrieving all data from the table, and then retrieving only specific columns (name and dept).

5. Update

Explanation: 
        Updating the city of the student with id 2 to 'Bangalore' using the WHERE clause.

6. Delete

Explanation:
        Deleting the record of the student with id 4 from the table using the WHERE clause.

Explanation:
        Removing all records from the 'students' table completely while keeping the table structure intact.
